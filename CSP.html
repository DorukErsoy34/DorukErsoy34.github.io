How compression works

Larger data file, the more that can be pulled out of it
Data compression methods:
Lossless:
Reduces the number of bits stored or transmitted while guaranteeing complete reconstruction of original data
Lossy:
Significantly reduces the number of bits stored or transmitted but only allows approximate reconstruction of original data.

Data Abstraction: Data abstraction refers to the process of simplifying complex real-world entities into 
manageable representations by focusing on their essential characteristics while hiding unnecessary details.
        
Key Terms:
- List: A collection of elements that are ordered and can be accessed by their position.
- Strings: Sequences of characters enclosed in quotation marks.
- Variable: A named storage location in computer memory that holds a value which can change during execution.
- Arithmetic Operators: Symbols for mathematical operations like addition, subtraction, etc.
- Iteration: Repetition of a set of instructions until a specific condition is met.
- MOD operator: Returns the remainder when one number is divided by another.
- Pseudocode: A simplified language using plain English to outline program logic.
- Selection: The process of making a decision based on a condition or criteria.
- Sequencing: The order in which code statements are executed.

Boolean variables can only represent true or false values, often used with relational and logical operators.
