1.5 
Casting and Range of Variables
Casting changes a variable’s type, e.g., turning a string into a number.
Example: Number("42") converts the string "42" to the number 42.
Range refers to the limits of values a type can hold (like how integers have a max size).
Quick takeaway: Casting ensures data is in the right form for calculations.
1.6 
Compound Assignment Operators
Shorthand operators update variables more quickly.
Example: x += 5 means x = x + 5; x *= 2 means x = x * 2.
Quick takeaway: Compound operators save time when updating variable values.
1.7 
Application Program Interface (API) and Libraries
APIs and libraries give access to ready-made functions.
Example: JavaScript’s Math library has functions like Math.sqrt(25) or Math.random().
Quick takeaway: APIs extend your language with powerful tools you don’t need to build yourself.
1.8
Documentation with Comments
Comments explain code for humans, ignored by the computer.
In JavaScript, // makes a single-line comment and /* ... */ makes a multi-line comment.
Quick takeaway: Comments make your code readable and maintainable.
1.9 
Method Signatures
A method (function) signature shows its name, parameters, and return type.
Example in JavaScript: function greet(name) { return "Hello " + name; } has name = greet, parameter = name, return = string.
Quick takeaway: Method signatures define how a function is used.
