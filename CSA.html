Casting and Range of Variables
Casting changes a variable’s type, e.g., turning a string into a number.
Example: Number("42") converts the string "42" to the number 42.
Range refers to the limits of values a type can hold (like how integers have a max size).
Quick takeaway: Casting ensures data is in the right form for calculations.

Shorthand operators update variables more quickly.
Example: x += 5 means x = x + 5; x *= 2 means x = x * 2.
Quick takeaway: Compound operators save time when updating variable values.

UNIT 2:

🔁 Iteration
Iteration means repeating a section of code multiple times.
In Java, this is done using loops such as:
for loops (when you know how many times to repeat)
while loops (when you repeat until a condition becomes false)
Example:
for (int i = 0; i < 5; i++) {
    System.out.println("Hello!");
}
This prints “Hello!” five times.

⚙️ Selection
Selection is when a program makes decisions based on conditions.
It allows the program to choose different paths of execution.
The most common selection structure is the if-else statement.
Example:
if (score >= 10) {
    System.out.println("You passed!");
} else {
    System.out.println("Try again!");
}
Here, the output depends on whether the score condition is true or false.

🧩 Boolean Expressions
A boolean is a data type that can hold only two values: true or false.
Boolean expressions are used in decision-making and control structures.
Example:
boolean isEven = (number % 2 == 0);
System.out.println(isEven);
This prints true if the number is even, or false otherwise.

🔄 If-Else Statements
The if-else structure is a key part of algorithm design because it introduces logic and flexibility.
It checks a condition, and if it’s true, it executes one block of code; otherwise, it executes another.

Example:
if (temperature > 30) {
    System.out.println("It's hot outside!");
} else if (temperature >= 20) {
    System.out.println("It's warm.");
} else {
    System.out.println("It's cold!");
}
This uses multiple conditions to select one of several possible outcomes.

2.4 Nested if Statements
An if statement inside another if (or else).
Used when a decision depends on another condition.
Example:
if (condition1) {
    if (condition2) {
        // both true
    } else {
        // condition1 true, condition2 false
    }
} else {
    // condition1 false
}
Nesting too deep can reduce readability.

2.5 Compound Boolean Expressions
Combine multiple boolean conditions using logical operators:
AND && → true if both are true
OR || → true if at least one is true
NOT ! → negates a boolean
Example:
boolean canRide = age >= 12 && height >= 48;
boolean alert = !isSafe || hasWarning;
Parentheses help clarify order of operations.
    
2.6 Comparing Boolean Expressions
Check if boolean expressions produce the same result.
== compares primitive values.
.equals() compares object contents.
Example:
boolean a = true;
boolean b = (5 > 3);
System.out.println(a == b); // true
For objects like Strings, == checks reference; use .equals() for content comparison.

2.7 While Loops
Definition: A while loop repeatedly executes a block of code as long as a condition is true.
Syntax:
while (condition) {
    // code to repeat
}
Key Points:
Condition is checked before the loop body runs (pre-test loop).
Infinite loop risk if condition never becomes false.
Can be used for unknown number of iterations.
Example:
int count = 0;
while (count < 5) {
    System.out.println(count);
    count++;
}

2.8 For Loops
Definition: A for loop is used when the number of iterations is known or predictable.
Syntax:
for (initialization; condition; update) {
    // code to repeat
}
Key Points:
Initialization: run once at the start.
Condition: checked before each iteration.
Update: executed after each iteration.
Can count up or down, or even use multiplication/division.
Example:
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

2.9 Implementing Selection and Iteration Algorithms
Selection (Decision Making)
Purpose: Choose between alternatives using if, else if, else, or switch.
Example:
int num = 10;
if (num > 0) {
    System.out.println("Positive");
} else if (num < 0) {
    System.out.println("Negative");
} else {
    System.out.println("Zero");
}
Iteration (Repeating Tasks)
Purpose: Repeat tasks using loops (while, for, do-while).
Common Patterns:
Counter-controlled iteration: repeat N times (use for).
Sentinel-controlled iteration: repeat until a special value is entered (use while).
Example (Counter-controlled):
for (int i = 1; i <= 5; i++) {
    System.out.println("Iteration " + i);
}
Example (Sentinel-controlled):
Scanner input = new Scanner(System.in);
int value = 0;
while (value != -1) {
    System.out.print("Enter a number (-1 to stop): ");
    value = input.nextInt();
}

2.11 Nested Iteration
Nested iteration = a loop inside another loop.
Outer loop controls overall repetitions, inner loop handles sub-tasks.
Commonly used for:
2D arrays or matrices
Printing patterns (e.g., triangles, squares)
String processing (looping over each character in multiple strings)
Example:

    }
}
2.12 Informal Run-Time Analysis
Estimates how long an algorithm takes without precise formulas.
Focus: How time grows as input size grows (efficiency).
Common patterns:
Loops → linear (O(n)), nested loops → quadratic (O(n^2))
Simple operations inside loops → constant time (O(1))
Helps compare algorithms and anticipate performance problems.
Example: Counting vowels in a string of length n → O(n) time. Nested loop over n x n → O(n^2).

3.7 Class Variables and Methods (static)

Keeping track of shared data (e.g., number of objects created)

3.8 Scope and Access
Scope determines where a variable or method can be used in a program.
Types of scope:
Local: inside a method/block – disappears after execution.
Instance: belongs to an object – accessible via that object.
Class (static): belongs to the class – shared by all objects.
Access modifiers control visibility:
public: accessible everywhere.
private: accessible only within the class.
protected: accessible in class, subclass, and same package.
default (no modifier): accessible within the package.

Searching Algorithms
Searching algorithms are used to find a specific value in a list or array.
Two common types:
1. Linear Search
- Checks each element one by one.
- Works on both sorted and unsorted lists.
- Slower for large lists.
Example:
int linearSearch(int[] arr, int target) {
    for(int i = 0; i < arr.length; i++) {
        if(arr[i] == target) {
            return i;
        }
    }
    return -1;
}
Time Complexity:
O(n)
2. Binary Search
- Works ONLY on sorted arrays.
- Repeatedly checks the middle element.
- Eliminates half of the list each step.
Example:
int binarySearch(int[] arr, int target) {
    int low = 0;
    int high = arr.length - 1;
    while(low <= high) {
        int mid = (low + high) / 2;
        if(arr[mid] == target) {
            return mid;
        }
        else if(arr[mid] < target) {
            low = mid + 1;
        }
        else {
            high = mid - 1;
        }
    }
    return -1;
}
Time Complexity:
O(log n)
Sorting Algorithms
Sorting algorithms arrange elements in a specific order
(usually ascending or descending).
Common algorithms:
1. Selection Sort
- Finds the smallest value.
- Swaps it with the first unsorted element.
Example:
void selectionSort(int[] arr) {
    for(int i = 0; i < arr.length - 1; i++) {
        int minIndex = i;
        for(int j = i + 1; j < arr.length; j++) {
            if(arr[j] < arr[minIndex]) {
                minIndex = j;
            }
        }
        int temp = arr[i];
        arr[i] = arr[minIndex];
        arr[minIndex] = temp;
    }
}
Time Complexity:
O(n²)
2. Insertion Sort
- Builds a sorted section of the array.
- Inserts each element into its correct position.
Example:
void insertionSort(int[] arr) {
    for(int i = 1; i < arr.length; i++) {
        int key = arr[i];
        int j = i - 1;
        while(j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
}
Recursion
Recursion is when a method calls itself.
Two important parts:
1. Base Case
The condition that stops recursion.
2. Recursive Case
The method calls itself with a smaller problem.
Example: Factorial
5! = 5 × 4 × 3 × 2 × 1
Recursive solution:
int factorial(int n) {
    if(n == 1) {
        return 1;   // base case
    }
    return n * factorial(n - 1);  // recursive call
}
Execution:
factorial(5)
= 5 * factorial(4)
= 5 * 4 * factorial(3)
= 5 * 4 * 3 * factorial(2)
= 5 * 4 * 3 * 2 * factorial(1)
= 120
Recursive Searching and Sorting
Searching and sorting algorithms can also be written using recursion.
Example: Recursive Binary Search
int binarySearch(int[] arr, int target, int low, int high) {
    if(low > high) {
        return -1;
    }
    int mid = (low + high) / 2;
    if(arr[mid] == target) {
        return mid;
    }
    else if(arr[mid] < target) {
        return binarySearch(arr, target, mid + 1, high);
    }
    else {
        return binarySearch(arr, target, low, mid - 1);
    }
}
Example Call:
binarySearch(arr, 25, 0, arr.length - 1)
